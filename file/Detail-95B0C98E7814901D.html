朝阳共青团二〇一六年工作要点

来源：    发布时间： 2016-05-19    作者： 团区委

2016年朝阳共青团将深入贯彻落实党的十八届五中全会、团中央十七届五中全会精神，深入学习贯彻习近平总书记系列重要讲话精神，按照区委十一届十一次全会、群团工作会议部署和团市委工作会议要求，牢牢把握朝阳区五方面阶段性特征，围绕朝阳区“五个全力”工作方向，以保持和增强团的政治性、先进性、群众性为目标，落实“凝聚青年、服务大局、当好桥梁、从严治团”四维工作格局，带领广大青年为朝阳区各项事业发展贡献青春力量。

【重点工作】

一、与时俱进，优化团的组织设置

1. 夯实完善基层组织体系。一是根据市、区群团改革要求，调整优化内设机构职能，提高工作效率。二是加强青年聚集新兴领域团建，重点向商务楼宇、园区、网络阵地等青年人气聚集的领域延伸组织体系。三是与工会、妇联等群团组织加强合作，完善“一统五联”工作机制，创新人、财、物保障机制，加强保障力度，在基层组织设置、成员发展、联系青年、活动方式等方面取得新突破。四是探索建立青年工作委员会试点，在青年较为集中的行业和领域，推动加强在同级党工委领导下的区域团工委建设，例如CBD管委会等功能区管委会或青年较为集中的委办局，将工作对象从28周岁以下的团员扩展到35周岁以下青年。五是在团市委领导下，开展“共青云”建设试点工作，建立团支部管理考核信息系统，推动团自身建设的互联网转型；制定简单、明确、可考核的团支部工作标准，形成一套重心下移、资源下沉、信息化管理、可量化考核的运行机制。在“职业+地域+兴趣”的三维组织体系下推进团支部建设，坚持团员标准，创新和严肃团的组织生活方式。

2. 着重强化学校共青团工作。巩固学校共青团基础性、战略性、源头性地位，严把团员入口关，保持适当的团学比例。适应教育体制改革，在巩固原有学校共青团工作基础上，重点加强民办学校等的团建工作。加强学校共青团和少先队工作，将学校团干部和少先队辅导员培训纳入区级团干部培训体系。

二、从严治团，加强团干部队伍建设

1. 多渠道加强团干部选拔配备。探索通过“专挂兼”方式配强团的领导班子，打破年龄、学历、身份、级别限制，从具有团青工作经验，基层群众工作经历的人员中选拔优秀青年担任团区委挂职或兼职副书记。开展高校学生到基层挂职工作，吸引优秀大学生到团区委部长助理和街乡（工）委书记助理岗位实习锻炼，引进新鲜血液补充团的工作力量，为高校学生提供社会实践渠道。

2. 建立代表广泛、民主参与的管理机制。结合朝阳区共青团第十二次团员代表大会召开契机，增强委员会委员和团代表的广泛性和代表性，提高常委会、全委会、团代表中基层一线团干部和团员的比例。3. 建立常态化联系服务基层机制。组织团区委机关干部通过全脱产和周驻点两种联系服务基层机制，一是让团区委机关干部中无街乡工作经历的同志到基层脱产挂职锻炼半年；二是让全体干部每人至少联系一街一乡每周驻点，推动落实至少一项重点难点工作，通过融入基层工作环境，了解青年实际需求，切实解决基层团组织和服务青年“最后一公里”的实际问题，实现青年干部“接地气”“争做青年友，不做青年官”的目标。

三、创新方式，开展新媒体建设工作

1. 以线上产品和线下活动加强青年思想引导。借助专业力量和市场力量，推动线上团属文化产品生产和创作，推介高质量微视频、动漫、歌曲等青年喜爱的文化精品；在重要节日、重大事件纪念日，开展弘扬传统文化和革命精神等形式多样的线下主题教育活动；结合热点话题、敏感事件发出团的声音，体现团的态度，展现团的作为，通过宣传产品及活动帮助青年树立理想信念的精神支柱，树立向上向善的价值追求。与市交通委等市区相关委办局合作，整合资源，联合发力，推出“朝阳青年”名片。

2. 创新新媒体工作方式。适应新形势，通过引进专业力量，购买专业服务等方式，加强青春朝阳微信、微博等平台建设。推进“青年之声•朝阳”平台建设，实现与共青团工作、社区青年汇的全面融合，以及团内团外、机关和基层的全面融合，完善平台的日常考核和沟通联络机制，建立涵盖团区委、街乡、社区、社区青年汇四个层级的线上专家库，倾听青年心声，回应青年诉求，维护青年权益，服务青年成长。稳定、优化朝阳区网宣员及骨干队伍，探索建立网宣员激励机制，依托微邦系统，完善网宣员调度机制。加强网络宣传专题培训，定期聘请知名讲师授课，全面提升网宣员新媒体运用和舆情应对能力，为积极抢占网络阵地，加强舆论引导夯实工作基础。

3. 精准定位用户群体。通过中小学生风采展示等小手拉大手活动，有效引导青年家长思想及行为。通过大学生社团开展针对性的兴趣爱好活动，有效引导高校学生思想及行为。通过创业就业活动，有效引导创业就业青年助力朝阳经济发展。通过优秀青年榜样力量宣传向广大青年传递青春正能量，弘扬社会风尚。

四、提质增效，强化就业创业工作

1. 完善机制，组建青年创业帮扶团队。联合区职能部门，发起以“凝聚创业人才、整合社会力量、优化服务资源、搭建服务平台、提高创业成活率”为主旨的区青年创业联盟。组建以社区青年汇社工和驻区高校就业创业干事为骨干的创业引导员队伍；以青联委员、天使投资人、风险投资机构为核心的创业投资人队伍；以青年企业家、高校学者、政府机构负责人、风险投资人为支撑的青年创业导师队伍；以律师协会为依托的青年创业公益法律顾问队伍。

2. 整合政策，强化青年创业服务。整合青年创业优惠政策，探索建立集区域产业政策、税收政策、人才培养政策为一体的朝阳区“三创”平台，利用线上平台广泛宣传、公开发布。探索建立以区财政经费为基础，社会募集、线上众筹为支撑，青年企业家、区青联委员积极参与的区级青年公益创业“种子基金”，为较为成熟的项目提供无偿小额启动资金支持。寻求区内更多专业投资机构为大学生创业企业提供融资服务产品。

3. 搭建平台，扩大创业服务的朋友圈。开展青年创业课堂及创业训练营，将政府、投融资机构、高校、创业者等各方需求进行有效对接，通过专家导师“多对一”专业分享及诊断、知识讲座、模拟实训等方式，提升创业人员的创新意识和创业能力。选拔优秀创业项目，入驻创业孵化基地，匹配导师“1对1”定向帮扶，对接资源。营造全区青年创新创业的浓厚氛围，引导青年想创业、敢创业、能创业。为津、冀在京青年返乡创业搭建平台，促进京津冀青年互动合作，助力非首都功能疏解。

4. 打造品牌，开展青年创新创业引领工程。深化青年就业创业见习基地项目，提升品牌影响力。围绕转变经济发展方式，构建“高精尖”经济结构，提高经济发展质量和效益中心任务，优化就业创业见习基地构成，以在朝阳区注册及纳税为优先条件，着重提升高新技术、金融服务、文化传媒等领域中的小微型企业的见习岗位比例。在初创型创业团队中设置见习岗位，推荐有创业意向青年参与见习，解决初创企业“招人难”的问题。为创业初期的团队举办“创业合伙人”招聘对接会，通过企业创始人的宣讲，招募企业合伙人及员工。以“国学讲堂进基地”、青令营、城市体验营等活动为开端，开展形式多样的培训交流活动，服务企业发展，拓宽服务青年范围。

五、科学谋划，巩固青年服务阵地建设

1. 科学合理调整社区青年汇布局。围绕加强和创新社会治理中心工作，不断完善社区青年汇运行机制。根据实际情况，在保证社区青年汇总数不变的前提下，对旗舰店、重点店、普通店进行调整；对于长期不能保障社区青年汇运行条件、多次考核落后的社区青年汇进行裁撤；尝试在众创空间、社创企业、社工之家建立社区青年汇，探索社区青年汇社会化运作模式。在全区选取2—3家设施完善、交通便利、学校聚集的场所建设青少年创新部落，做好青少年课外科技创新教育引领。

2. 加强社区青年汇队伍建设。继续开展专职社工知行力培训工作，完成注册社工师和心理咨询师考证培训。开展每月两期的青少年事务社工实务沙龙，邀请专家、督导、优秀社工以成长小组的形式，提升社工实务专业化水平。完善月度督导例会制度和督导考评退出机制，加强社工日常管理。积极引入高校专业人才到社区青年汇实习，提供专业理论指导。

3. 打造项目化品牌活动体系。联合专业机构自主开发和引入一批优秀活动项目，建立区级优秀项目库。打造各社区青年汇品牌活动，每月固定一天为社区青年汇公益日，开展志愿服务工作。探索建立活动标准化流程，确保青年服务体验效果。

4. 维护青少年合法权益。充分发挥区预青组、未委会统筹协调作用，深化法治宣传教育、星光自护等活动内涵，做好重点青少年等弱势群体的服务管理和预防犯罪工作。针对辖区“五需”、涉诉、流动人员子女等重点青少年群体的不同需求，依托专业社会组织机构，进一步加强合适成年人、青少年维权专家队伍建设，开展专业化个案帮扶，切实履行重点青少年群体个案帮扶第二轮试点单位职责。成立助困助学众筹基金，利用微信募捐等新模式，凝聚社会各界力量帮扶弱势青少年群体。

【常态化工作】

一、夯实团建基层基础

1. 确保区域化团建工作常态化。严格落实健全街乡团（工）委班子、召开区域团组织工作会等任务，确保机制常态化，完善以街乡团（工）委为核心的条块结合、资源共享、广泛联系、共同参与、相互促进的区域化青年工作格局。结合属地特点、青年需求，深化“一街一品、一社一品”活动，使活动品牌化、特色化、规模化。依托社区青年汇，组织策划形式多样、内容丰富的活动项目，使基层团组织成为服务青年的“共青团门店”。

2. 加强基层工作资金保障。坚持人往基层走、钱给基层用
